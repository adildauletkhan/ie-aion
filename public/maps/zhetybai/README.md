# Карты месторождения Жетыбай (GeoJSON)

## 📍 Координаты: 43.84° с.ш., 51.81° в.д.

## 📁 Слои карты:

### 1. **field_boundary.geojson** - Контуры месторождения
- Внешний контур (145.3 км²)
- Внутренний контур высокопродуктивной зоны (78.5 км²)

### 2. **thickness_isolines.geojson** - Изолинии толщины
- 5 изолиний: 10м, 15м, 20м, 25м, 30м
- Цвета: зелёный → жёлтый → красный

### 3. **porosity_isolines.geojson** - Изолинии пористости  
- 5 изолиний: 18%, 20%, 22%, 24%, 26%
- Цвета: серый → голубой → синий

### 4. **wells.geojson** - 13 скважин
- 7 добывающих (101-107)
- 4 нагнетательных (201-204)
- 2 наблюдательных (301-302)

### 5. **drainage_zones.geojson** - 5 зон дренирования
Синхронизировано с БД

### 6. **faults.geojson** - 3 тектонических разлома

## 🎨 Порядок отображения (снизу вверх):
1. Контуры → 2. Разломы → 3. Изолинии → 4. Зоны → 5. Скважины

## 📊 Формат: GeoJSON (WGS 84, EPSG:4326)
